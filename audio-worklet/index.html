AudioWorklet

AudioWorklet is currently available on Chrome behind the experimental flag. Use the following command line option to activate the feature.

--enable-blink-features=Worklet,AudioWorklet

Or you can enable Experimental Web Platform Features in Chrome's flag setting. Note that this setting enables all the experimental features in browser. Copy and paste the following URL in Chrome.

chrome://flags/#enable-experimental-web-platform-features
